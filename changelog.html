Changelog

v2.0.9 Apr 2026 Final

Project Discontinued — Final Release

  • sunsetNIX is shutting down. Shadowstrap was a proof of concept — a test of what a Roblox bootstrapper could actually be. That question has been answered. No further updates will be released.
  • addSunset dialog — a discontinuation notice is shown on every user-facing launch, with a direct Uninstall Now button that triggers the built-in uninstaller to restore the default Roblox launcher.
  • removeAuto-update permanently disabled — the update check is removed at the code level. v2.0.9 is the last binary that will ever be distributed.
v2.0.8 Apr 2026 Stable

Cleanup — Removed Features

  • removeGame Profiles — per-game FastFlag profile tab and all related logic removed. The feature had reliability issues and added complexity without enough practical use.
  • removeInventory Viewer — player collectible inventory lookup tab removed. Roblox API changes made the data unreliable.
  • removeHone.gg Integration — auto-launch integration removed from the Integrations page. The integration had no meaningful adoption.
  • removeCustom Death Sound — in-app death sound file picker removed from Mods page. You can still place any ouch.ogg directly in the Modifications folder manually.
v2.0.7 Apr 2026 Stable

Hotfix — Settings Crash

  • fixApp crashed immediately when opening the Settings window — XamlParseException: GameChat20 is not a valid value for SymbolRegular. Icon name corrected to a valid SymbolRegular enum value.
v2.0.6 Apr 2026 Stable

Per-Game FastFlag Profiles & Community Sharing

  • addGame Profiles — new sidebar tab. NIX automatically writes a custom FastFlag set when you join a specific game and restores your original flags on leave.
  • add8 built-in game presets: The Strongest Battlegrounds, Blox Fruits, Murder Mystery 2, Jailbreak, Arsenal, Da Hood, Brookhaven RP, Pet Simulator 99 — each tuned for max FPS and reduced visual overhead.
  • addCustom profile CRUD — add, edit, and delete your own per-game flag sets with a place ID lookup that auto-fills the game name.
  • addProfile toggle — enable or disable any built-in or custom profile individually without deleting it.
  • addUser profiles override built-in profiles for the same Place ID, giving full per-game customisation without losing the defaults.
  • addCommunity flag sharing — Share preset and Browse Community buttons in the FastFlag Editor export or import flag bundles as JSON for GitHub pull requests.
  • addcommunity/flags.json added to the repo — community-contributed FastFlag presets live here.
  • addImport Community Profile — paste a JSON blob from GitHub directly into the Game Profiles tab to add a profile without manual entry.
v2.0.5 Apr 2026 Stable

FastFlag Editor Tab & Inventory Viewer

  • addFastFlag Editor is now a top-level sidebar tab — no longer hidden behind the presets page or gated by Roblox Studio.
  • addNew Inventory tab — enter any Roblox username or user ID to view their public collectible inventory with item thumbnails, names, RAP, and serial numbers.
  • addTotal RAP summary card shown at the top of the inventory results.
  • addCursor-based Load More pagination for large inventories.
  • addPrivate inventory detection — clear error message instead of an empty list.
v2.0.4 Apr 2026 Stable

Hotfix — Session Summary & Rendering Modes

  • fixSession Summary now shows the correct game — it was always displaying the first game of the session because the History list is ordered newest→oldest and the wrong end was being read.
  • changeRemoved OpenGL and Vulkan rendering mode options (reverted from v2.0.3).
v2.0.3 Apr 2026 Stable

FastFlags Fix & Rendering Modes

  • fixPerformance Presets now actually apply — ClientAppSettings.json was never written to disk when clicking Apply. All flag setters now call Save() immediately.
  • fixFPS Counter toggle now persists — enabling FFlagDebugDisplayFPS is written to disk on toggle so the overlay appears in-game without needing a separate Save click.
  • fixMSAA, Rendering Mode, Texture Quality, and Display Scaling dropdowns/toggles all save immediately on change.
  • addActive preset highlight — the card for the currently applied preset glows purple on the FastFlags page, persisted across sessions.
  • addOpenGL rendering mode — FFlagDebugGraphicsPreferOpenGL exposed in the Rendering Mode dropdown.
  • addVulkan rendering mode — FFlagDebugGraphicsPreferVulkan exposed in the Rendering Mode dropdown.
v2.0.2 Apr 2026 Stable

Server Browser & Session Summary

  • addServer Browser — tray menu item that lets you browse open public servers for any game by Place ID. Shows player count, ping, FPS, and estimated region for each server instance.
  • addJoin specific server directly from the browser list via roblox://experiences/start?placeId=…&gameInstanceId=…
  • addHide Full filter and cursor-based Load More pagination in the server list
  • addSession Summary popup — shown automatically after leaving a game. Displays game thumbnail, name, time played, join/leave times, and server region (loaded async via ipinfo.io).
  • addRejoin button in session summary jumps back into the same server
  • addShowSessionSummary setting to disable the popup if unwanted (default on)
v2.0.1 Apr 2026 Stable

Hotfix & version bump

  • fixCrash on Mods tab load caused by invalid WPF UI icon name (MusicNote1Add24 → MusicNote124)
  • changeVersion corrected from leftover 2.11.0 Bloxstrap value to 2.0.1
  • changeAuto-updater now correctly compares against v2.0.x tags — users on v2.0.0 are prompted on next launch
v2.0.0 Apr 2026 Stable

NIX — initial public release

  • addFull rebrand from Bloxstrap to NIX (Shadowstrap) — new name, purple #8B5CF6 theme, updated assets throughout
  • addPerformance Presets — one-click FastFlag bundles for Default, Balanced, and Max Performance modes
  • addProcess Priority Boost — elevate Roblox to Above Normal, High, or Realtime CPU scheduling class
  • addMulti-Instance support — closes the RobloxSingletonMutex via P/Invoke so multiple windows can run simultaneously
  • addRAM Cleaner — trims other processes' working sets using SetProcessWorkingSetSize before Roblox launches
  • addFPS Counter toggle — FFlagDebugDisplayFPS preset wired into the FastFlags page
  • addAuto-Rejoin — Watcher process monitors Roblox exit and relaunches the original roblox:// URL after 5 seconds
  • addCustom Death Sound — file picker for .ogg, .mp3, .wav; copies to Modifications/content/sounds/ouch.ogg
  • addHone.gg integration — auto-detects and launches Hone before Roblox with a 2.5s initialization window
  • addGitHub Actions CI pipeline — builds and publishes release artifacts on version tags
  • addFull GitHub wiki — 17 pages covering all features, settings, troubleshooting, and modding
  • changeWinget package identifier updated from pizzaboxer.Bloxstrap to binx-ux.Shadowstrap
  • changeAll namespaces, assembly names, and string resources renamed from Bloxstrap to Shadowstrap
