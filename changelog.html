Changelog

Every version, every change. Most recent first.

v2.0.2 Apr 2026 Current

Server Browser & Session Summary

  • addServer Browser — tray menu item that lets you browse open public servers for any game by Place ID. Shows player count, ping, FPS, and estimated region for each server instance.
  • addJoin specific server directly from the browser list via roblox://experiences/start?placeId=…&gameInstanceId=…
  • addHide Full filter and cursor-based Load More pagination in the server list
  • addSession Summary popup — shown automatically after leaving a game. Displays game thumbnail, name, time played, join/leave times, and server region (loaded async via ipinfo.io).
  • addRejoin button in session summary jumps back into the same server
  • addShowSessionSummary setting to disable the popup if unwanted (default on)
v2.0.1 Apr 2026 Stable

Hotfix & version bump

  • fixCrash on Mods tab load caused by invalid WPF UI icon name (MusicNote1Add24 → MusicNote124)
  • changeVersion corrected from leftover 2.11.0 Bloxstrap value to 2.0.1
  • changeAuto-updater now correctly compares against v2.0.x tags — users on v2.0.0 are prompted on next launch
v2.0.0 Apr 2026 Stable

NIX — initial public release

  • addFull rebrand from Bloxstrap to NIX (Shadowstrap) — new name, purple #8B5CF6 theme, updated assets throughout
  • addPerformance Presets — one-click FastFlag bundles for Default, Balanced, and Max Performance modes
  • addProcess Priority Boost — elevate Roblox to Above Normal, High, or Realtime CPU scheduling class
  • addMulti-Instance support — closes the RobloxSingletonMutex via P/Invoke so multiple windows can run simultaneously
  • addRAM Cleaner — trims other processes' working sets using SetProcessWorkingSetSize before Roblox launches
  • addFPS Counter toggle — FFlagDebugDisplayFPS preset wired into the FastFlags page
  • addAuto-Rejoin — Watcher process monitors Roblox exit and relaunches the original roblox:// URL after 5 seconds
  • addCustom Death Sound — file picker for .ogg, .mp3, .wav; copies to Modifications/content/sounds/ouch.ogg
  • addHone.gg integration — auto-detects and launches Hone before Roblox with a 2.5s initialization window
  • addGitHub Actions CI pipeline — builds and publishes release artifacts on version tags
  • addFull GitHub wiki — 17 pages covering all features, settings, troubleshooting, and modding
  • changeWinget package identifier updated from pizzaboxer.Bloxstrap to binx-ux.Shadowstrap
  • changeAll namespaces, assembly names, and string resources renamed from Bloxstrap to Shadowstrap
