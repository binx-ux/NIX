Setup & Executors

Troubleshooting
First, confirm you are using a Level 7 executor — lower-level executors will not be able to run the full script. Detach the executor, reattach it, and execute again. If it still does not appear, try a different game and then try again. If the issue persists, join the Discord and post your error logs.
Copy the full error message from your executor console and bring it to the Discord support channel. Most errors are executor-side and resolve by switching to Xeno or ensuring your executor is up to date.
Not all games have aimbot support. Check the Games page to see which titles have full aimbot enabled. If your game is not listed, it uses the universal feature set which may not include aimbot depending on how that game handles characters.
The hub fetches Vertex and Phantasm from their own external URLs. If those scripts are down or their source URLs have changed, the auto-load will fail. Check the Discord for any announcements about external script outages.
