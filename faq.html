Safety & Bans

Installation & Setup
Click More info then Run anyway. SmartScreen warns about unsigned executables — NIX isn't code-signed yet. The binary is built directly from the public source on GitHub. If you're unsure, you can build it yourself from source.
NIX installs to %LocalAppData%\Shadowstrap\. It registers itself as the handler for the roblox:// URL protocol, creates a Modifications folder for mods, and writes a settings file. Nothing is installed system-wide.
Yes. NIX handles Studio launches too. Player-only features like Discord Rich Presence and Activity Tracking won't apply during Studio sessions, but launching Studio from the website or Start Menu works normally.
Features
Yes. On every launch NIX calls the GitHub releases API, compares the installed version against the latest tag, and prompts you if a newer version is available. The update check respects your preference — you can have it notify you or update silently.
Roblox normally uses a Windows mutex named RobloxSingletonMutex to prevent a second instance from starting. When Multi-Instance is enabled, NIX enumerates the handles of the running Roblox process using NtQuerySystemInformation and closes that mutex handle before launching the next instance.
Open NIX → Settings → Mods → Open Mods Folder. The folder mirrors the Roblox content directory structure — files placed here override the matching files in the game installation. For example, content/sounds/ouch.ogg replaces the death sound. The Custom Death Sound option in Settings handles this automatically.
NIX accepts .ogg, .mp3, and .wav. When you pick a file, it's converted and saved as ouch.ogg in the mods folder automatically. Removing the sound via the button deletes that file and restores the default.
Activity Tracking reads the Roblox log file to detect which game you joined, how long you were there, and what server you were on. This data is stored locally — it's never uploaded. The only external call is an optional IP geolocation lookup via ipinfo.io to show server region in Discord, and that only sends the server's IP, not yours.
Uninstalling
Open NIX → Settings → NIX tab → Uninstall. This removes NIX and restores the default Roblox bootstrapper. Your Roblox installation (%LocalAppData%\Roblox\) is left completely intact.
Delete the folder %LocalAppData%\Shadowstrap\. Then reinstall Roblox from roblox.com to restore the default bootstrapper and protocol handler. Your saved games and Roblox data are unaffected.
