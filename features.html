⚡ Performance

Performance Presets

One-click FastFlag bundles that measurably improve FPS and reduce input latency without manual editing.

  • Default, Balanced, and Max Performance profiles
  • Tuned for real hardware — not benchmarks
  • Adjusts renderer, lighting, texture quality
  • Active preset highlighted in the UI
🏳️ Performance

FastFlag Editor

Full JSON editor built directly into settings. No digging through AppData folders or restarting Roblox after every change.

  • Search and filter any FastFlag by name
  • Toggle flags with a single click
  • Pin frequently used flags to the top
  • Changes apply on next Roblox launch
🧹 Performance

RAM Cleaner

Trims the working sets of background processes before Roblox starts, freeing physical RAM from the first frame.

  • Runs once per launch, transparently
  • No impact on other running apps long-term
  • Works alongside process priority boost
🚀 Performance

CPU Priority Boost

Elevate Roblox's CPU scheduling class so it wins contention against background processes during load spikes.

  • Above Normal, High, or Realtime modes
  • Applied immediately after launch
  • Reduces frame time variance under load
🌐 Performance

Hone.gg Integration

Launch the Hone network optimizer before Roblox connects, reducing packet loss and improving route stability.

  • 2.5 s warm-up window before game launch
  • Toggled on/off from integrations settings
  • Requires a Hone account to use
🔁 Automation

Auto-Rejoin

NIX watches the Roblox process. When it exits unexpectedly, it waits 5 seconds and relaunches the original server URL automatically.

  • Preserves the exact server you were on
  • Configurable delay (default 5 s)
  • Works with multi-instance sessions
🎯 Automation

Per-Game Profiles

Automatically write a custom FastFlag set when you join a specific game, then restore your default flags when you leave.

  • 8 built-in presets for popular games
  • Create unlimited custom profiles
  • Toggle each profile individually
  • Import/export as JSON for community sharing
🎮 Social

Discord Rich Presence

Your Discord status shows exactly what you're playing — game name, server type, time elapsed, and server region.

  • Game thumbnail shown in Discord
  • Server region via optional IP geolocation
  • Elapsed time tracked per session
  • Works with Private Servers and VIP servers
📊 Social

Activity Tracking

Every game session is logged locally — game name, playtime, server, and date. All stored on your machine, never uploaded.

  • Full session history in the tray menu
  • Session summary popup after leaving a game
  • One-click rejoin from session summary
  • Zero external data transmission
🔍 Social

Server Browser

Browse open public servers for any game by Place ID, sorted by ping, player count, and region. Join directly from the list.

  • Ping, FPS, region, and fill rate shown
  • Direct join with one click
  • Works for any public Roblox game
💎 UI & Tools

Inventory Viewer

Look up any player's public collectible inventory — item thumbnails, RAP, serial numbers, and acquisition dates.

  • Search by Roblox username
  • RAP (Recent Average Price) displayed per item
  • Serial number and series information
  • Reads from public Roblox API only
🪟 UI & Tools

Multi-Instance

Run as many simultaneous Roblox windows as your machine handles. NIX releases the singleton mutex before each launch.

  • No scripts or third-party tools required
  • Uses NtQuerySystemInformation to close mutex
  • Each instance runs independently
  • Discord RPC updates for the active instance
🖼️ Mods

Mod Support

Override any Roblox content file by dropping replacements into the Modifications folder. Mods persist through Roblox updates.

  • Mirrors the Roblox content directory layout
  • Built-in custom death sound picker
  • Accepts .ogg, .mp3, and .wav for sounds
  • Custom cursors and fonts supported
📋 UI & Tools

Session Summary

A popup after every session showing game thumbnail, total playtime, server region, and a one-click rejoin button.

  • Displayed automatically on Roblox exit
  • Rejoin puts you back on the same server
  • Dismiss or keep open — your choice
Deep Dive

Performance Presets,
explained.

The default Roblox client ships with conservative flag values that prioritize compatibility over performance. NIX's presets push the flags that actually matter.

  • Max Performance disables shadow maps, reduces texture resolution, and enables the Metal (fast-path) renderer where supported
  • Balanced keeps visual quality while removing the flags that cause frame time spikes on mid-range hardware
  • All presets are read-only starting points — you can customize any flag on top of them via the full editor

Deep Dive

Per-Game Profiles.
Automatic.

Different games need different flags. A combat game needs low-latency settings. A showcase needs high quality rendering. NIX handles the switch automatically on game join.

  • NIX reads the Roblox log to detect the Place ID the moment you join
  • The matching profile's flags are written to disk and a rejoin prompt appears so they take effect immediately
  • When you leave, your original flags are restored automatically — no manual switching
  • Built-in presets cover 8 of the most popular games, tuned by community testing
