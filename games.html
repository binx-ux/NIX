Universal Support — All Other Games

Even if your game is not in the list above, Binxix Hub still runs. Universal features including infinite jump, noclip, auto sprint, click teleport, speed hack, time freeze, and auto collect work in any Roblox game. Only ESP and game-specific modules require a dedicated Place ID entry.

Want your game added? Join the Discord and request it. If there is enough demand and the game is supportable, it will be included in the next update.
