// a note from the builder

Shadowstrap started as a fork and became something more — a personal experiment in what a Roblox bootstrapper could look like if someone actually cared. Performance tuning. A proper UI. Features the default launcher should have shipped years ago.

It was never a business. It was never trying to be. It was a test of what's possible in a weekend, a week, a month — and it turns out the answer is: quite a lot.

The source code stays public. Everything built here is documented, readable, and available to anyone who wants to learn from it, fork it, or take it further. The experiment is over. The work isn't wasted.
