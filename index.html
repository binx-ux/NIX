02 — Features

Built for people
who actually play.

Every feature came from someone asking for it. Nothing is here to look good on a list.

⚡
PERF_PRESETS

Performance Presets

One-click FastFlag bundles — Default, Balanced, Max Performance. Tuned for real hardware, not spec sheets. The FPS difference is measurable.

FastFlags FPS Boost Renderer
🎮
DISCORD_RPC

Discord Rich Presence

Shows which game you're in, which server, elapsed time, and server region if you enable location lookup. Works automatically once you enable it.

🔁
RELIABILITY

Auto-Rejoin

NIX watches the Roblox process. When it exits, it waits 5 seconds and relaunches into the same server automatically.

🪟
MULTI_INST

Multi-Instance

Run as many Roblox windows as your machine handles. NIX closes the singleton mutex each time so instances don't block each other.

Mutex unlock
🖼️
MODS

Mod Support

Drop files into the Modifications folder — cursors, fonts, sounds. Override any game asset without touching the Roblox install. Mods survive updates.

🏳️
FASTFLAGS

FastFlag Editor

Full JSON editor built into settings. Search, toggle, pin any FastFlag without digging through config files or restarting Roblox.

🧹
MEMORY

RAM Cleaner

Trims other processes' working sets before Roblox starts. More physical RAM available from the first frame.

🚀
SCHEDULING

Priority Boost

Set Roblox's CPU scheduling class to Above Normal, High, or Realtime. It wins CPU contention during load spikes.

03 — Install

Three steps.
That's it.

NIX installs itself and registers as your Roblox launcher. You don't touch anything after that.

01

Download

Click the button below. You get a single .exe file. No zip, no installer wizard, no bundled garbage.

02

Run it

Windows SmartScreen warns you because the binary isn't signed yet. Click More info → Run anyway. NIX sets itself up and registers the roblox:// protocol handler.

03

Open any Roblox game

Hit Play on the Roblox site or from the Start Menu. NIX intercepts the launch. Configure settings, mods, and integrations from the tray icon.

04 — FAQ

Common questions.

Is this safe to use?

Yes. The full source is on GitHub and auditable by anyone. NIX doesn't touch the game client — it replaces the bootstrapper that launches Roblox, not Roblox itself.

Will I get banned?

No. NIX doesn't interact with the game engine the way exploits do. It's a launcher. The same reason users of Bloxstrap (the upstream project) aren't banned.

Does it auto-update?

Yes. On every launch NIX checks GitHub releases for a newer version. If one exists you'll get a prompt — or it updates silently based on your preference.

How do I uninstall?

Settings → NIX tab → Uninstall. Removes NIX and restores the default Roblox bootstrapper. Your Roblox installation is completely untouched.
