Notice — Development
Development Pause — Updates On Hold
March 26, 2026  ·  Effective immediately

After careful consideration, Binxix Hub development is being paused indefinitely. Continuing to develop without a clear creative direction would compromise the quality and standards this project has been built on. Rather than releasing updates that don't meet expectations, we're stepping back responsibly.

To all testers and contributors:

  • You will receive your scheduled payment on the 1st — your time and effort will be honored
  • After that payment, you are no longer required to continue testing — there won't be further testing cycles for the foreseeable future
  • Your contributions have been essential in shaping this project and are deeply appreciated

Open Source Transition: The project will be transitioned to an open source model and made publicly available on GitHub. The community is free to access, modify, and build upon the codebase. If you redistribute:

  • All original developer names must be removed before redistribution
  • The project must be renamed prior to publication

This pause does not necessarily represent a permanent conclusion — but for now, development stands still.

HIGH — Security
Webhook Attack & False Stealer Claims
March 24, 2026  ·  Resolved

We became aware of claims that Binxix Hub contains a "brain rot auto-stealer" — a webhook-based credential harvester. This is false. After a full review of the codebase, there is no exfiltration code, no credential harvesting, and no auto-stealer of any kind. The claim originates from a tampered third-party fork, not from our official loadstring or GitHub repo.

Binxix Hub is and always has been free and open source — every line is auditable on GitHub. If what you're running doesn't match our repo, you are running a modified version.

  • Always load from the official GitHub raw URL only
  • Never run scripts from random Discords or pastebin links
  • If you are distributing a malicious fork of our code, we will take action — DMCA, platform reports, public disclosure
Official source: github.com/binx-ux/airhub-binxix-v7
