What's Inside

Core Features

Aimbot

Silent, smooth, and configurable. Lock onto the nearest target with full control over FOV, sensitivity, and aim part.

  • Adjustable FOV circle
  • Head / Torso / Root targeting
  • Smooth aim interpolation
  • Toggle key support
  • Team check
Wall Hack ESP

CS:GO-styled ESP renders players through walls in real time with skeleton, boxes, tracers, and labels.

  • Full body skeleton
  • Bounding box
  • Tracers (Bottom / Center / Mouse)
  • Name and distance labels
  • Health bar overlay
Crosshair

Custom on-screen crosshair overlay with full color, size, gap, and center dot configuration.

  • Custom size and gap
  • Color picker
  • Center dot toggle
  • Outline support
Movement

Override your character's default movement limits. Set walkspeed and jump power to any value instantly.

  • WalkSpeed control
  • JumpPower control
  • Reset to defaults
Custom UI

Drag-anywhere GUI with tabbed layout. Open and close with Right Shift. Fully configurable keybind.

  • Tabbed settings panel

  • Config save and load
  • Reset all button

Quick Start

How to Use

01
Get an Executor

You need a Roblox script executor. Synapse X, KRNL, Electron, and Fluxus all work. Make sure it supports game:HttpGet.

02
Copy the Loadstring

Click the loadstring below to copy it to your clipboard.

loadstring(game:HttpGet("https://raw.githubusercontent.com/binx-ux/airhub-binxix-v6/main/source/Main.lua"))()
03
Launch and Inject

Start Roblox, enter any supported game, attach your executor, paste the loadstring, and hit execute.

04
Open the GUI

Press Right Shift to toggle the panel open and closed. You can change the keybind in the Settings tab inside the GUI.
